# Проект №11: IVI - iOS Functional & UI Testing (Critical Flows)

## Описание
Проект содержит набор регрессионных и функциональных тест-кейсов для мобильного приложения **IVI (iOS)**.  
В рамках проекта выполнено тестирование критических пользовательских сценариев и проверка исправления багов, затронувших различные модули приложения: навигацию по контенту, чат поддержки, работу с аккаунтами, обработку deeplink-ов, UI-консистентность и рекламные интеграции.

Проект оформлен в формате QA-портфолио с детализированными тест-кейсами и указанием на ожидаемый результат по задачам из трекера.

## Цель тестирования
- Проверить корректность отображения и навигации в блоке "Сезоны и серии".
- Верифицировать стабильность работы приложения при переходе в раздел чата поддержки.
- Проверить полный жизненный цикл детского аккаунта: от действий внутри подписки до его удаления из родительского профиля.
- Проверить обработку некорректного deeplink-а и отображение соответствующей ошибки.
- Убедиться в отсутствии новогодней айдентики после отката праздничного обновления.
- Проверить соответствие UI-макетам на лендинге подписки.
- Проверить корректность работы интерактивной рекламы в плеере.
- Проверить доступность и корректность отображения раздела "Все подписки".
- Верифицировать состояние A/B-тестов в режиме разработчика.

## Тип тестирования
- Functional testing
- UI testing
- Regression testing
- Smoke testing
