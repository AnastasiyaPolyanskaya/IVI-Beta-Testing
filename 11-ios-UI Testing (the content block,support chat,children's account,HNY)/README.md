# Проект №11: IVI -iOS Functional & UI Testing (Critical User Flows)

## Описание
Проект содержит набор регрессионных и функциональных тест-кейсов для мобильного приложения **IVI (iOS)**.  
В рамках проекта выполнено тестирование критических пользовательских сценариев и проверка исправления багов, затронувших различные модули приложения: навигация по контенту, чат поддержки, работу с аккаунтами, отображение новогодней айдентики, UI-консистентность и A/B-тесты.

Проект оформлен в формате QA-портфолио с детализированными тест-кейсами и указанием на ожидаемый результат по задачам из трекера.

## Цель тестирования
- Проверить корректность отображения и навигации в блоке "Сезоны и серии"
- Верифицировать стабильность работы приложения при переходе в раздел чата поддержки
- Проверить полный жизненный цикл детского аккаунта: от действий внутри подписки до его удаления из родительского профиля
- Убедиться в отсутствии новогодней айдентики после отката праздничного обновления
- Проверить соответствие UI-макетам на лендинге подписки
- Проверить корректность отображения и работы A/B-тестов в режиме разработчика
- Проверить видимость всех элементов интерфейса в критических разделах приложения

## Тип тестирования
- Functional testing
- UI testing
- Regression testing
- Smoke testing

- Устройство: iPhone 14 Pro Max
- ОС: iOS 18.6.2
- Версия приложения IVI: 26.1.2
- Тип аккаунта для тестирования: Взрослый (основной), Детский (временный), Неавторизованный пользователь
