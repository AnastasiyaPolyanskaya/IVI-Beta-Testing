# bug-reports

В данной папке собраны отчеты о дефектах (bug reports), обнаруженных в ходе тестирования проекта №11.

## Содержание


### Категории дефектов
1. **UI/UX проблемы** (некорректное отображение, верстка)
2. **Функциональные дефекты** (нарушение логики работы)
3. **Проблемы навигации** (некорректные переходы между экранами)
4. **Контентные проблемы** (некорректный контент, изображения)
5. **Визуальные дефекты** (несоответствие макетам)
