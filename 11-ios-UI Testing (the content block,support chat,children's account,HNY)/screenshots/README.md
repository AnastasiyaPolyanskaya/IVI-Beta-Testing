# Screenshots

В данном разделе собраны скриншоты, полученные в ходе функционального тестирования мобильного приложения **IVI (iOS)**.

Скриншоты используются как подтверждающие артефакты выполнения тест-кейсов проекта №11 и демонстрируют корректное поведение приложения при проверке критических пользовательских сценариев.

## Что представлено на скриншотах

### 1. UI & Навигация
- Корректное выделение активной серии в блоке "Сезоны и серии" (IOS-18421)
- Верстка лендинга подписки с правильными отступами (IOS-18453)
- Раздел "Все подписки" в профиле пользователя (IOS-17915)

### 2. Работа с аккаунтами
- Экран продления подписки в детском аккаунте (IOS-18379)
- Диалог подтверждения удаления детского аккаунта (IOS-184799)
- Экран "Мои аккаунты" после успешного удаления

### 3. Обработка ошибок и интеграции
- Экран ошибки "Что-то пошло не так" при переходе по некорректному deeplink (IOS-18342)
- Чат поддержки в стабильном состоянии (IOS-18478)
- Режим разработчика со списком A/B-тестов (IOS-17497)

### 4. Контент и реклама
- Интерфейс приложения без новогодней айдентики (IOS-18458)
- Плеер с рекламным блоком (A/B-тест "Клик по области видео в рекламе") (IOS-18274)

### 5. Критические переходы
- Экран подписки для неавторизованного пользователя
- Процесс переключения между взрослым и детским аккаунтом
- Различные состояния UI при выполнении операций
