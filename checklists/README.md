# Checklists — IVI Beta Testing

В этом разделе хранятся чек-листы, созданные в рамках функционального и исследовательского тестирования сервиса IVI.

Чек-листы помогают быстро оценить стабильность продукта, проверить ключевые пользовательские сценарии и убедиться, что основные функции работают корректно на разных устройствах.

## Что содержится в этой папке

- чек-лист авторизации и регистрации
- чек-лист работы видеоплеера
- чек-лист каталога фильмов
- чек-лист UI/UX элементов
- чек-лист мобильной версии
- дополнительные чек-листы по отдельным сценариям (по мере обновления проекта)

## Назначение раздела

Чек-листы позволяют ускорить регрессионное тестирование, снизить риски пропуска дефектов и создать единый стандарт проверки функциональности сервиса IVI.
