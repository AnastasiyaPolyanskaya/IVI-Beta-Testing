**ID:** ивпркд10.1  
**Название:** Проверка появления ассерта после удаления загруженного контента  
**Предусловия:**
Пользователь с активной подпиской
Зайти на главную страницу
Перейти на карточку сериального контента, доступного для загрузки. У сериала должно быть более 8 серий в сезоне
Перейти к загрузке сериала

**Шаги:**
Скачать 1 серию сериального контента ("Постучись в мою дверь")
Выгрузить приложение из памяти
Зайти в приложение и открыть загрузки
Удалить загруженный контент

**ОР:** Ассерт-Сообщение об ошибке на Android, имеет кнопки "Продолжить без ассертов" и "Сохранить лог"
