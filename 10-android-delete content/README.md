Описание: Тестирование функционала удаления загруженного контента и поведения попапов при переустановке приложения

Проект содержит набор тест-кейсов для мобильного приложения IVI (android).

В рамках проекта выполнено функциональное и UI-тестирование

Проект оформлен в формате QA-портфолио с тестовой документацией и визуальными артефактами (скриншоты).

Цель тестирования: Проверить корректность работы удаления контента и отображения системных попапов

Тип тестирования: Функциональное тестирование, UI-тестирование, Регрессионное тестирование

## Тест-кейсы (версия 26.1.1)

### UI - Главная страница

ID: ивпркд10.1
Название: Проверка появления ассерта после удаления загруженного контента
Предусловия:
1. Пользователь с активной подпиской
2. Зайти на главную страницу
3. Перейти на карточку сериального контента, доступного для загрузки. У сериала должно быть более 8 серий в сезоне
4. Перейти к загрузке сериала

Шаги:
1. Скачать 1 серию сериального контента ("Постучись в мою дверь")
2. Выгрузить приложение из памяти
3. Зайти в приложение и открыть загрузки
4. Удалить загруженный контент

Ожидаемый результат (ОР): Ассерт-Сообщение об ошибке на Android, имеет кнопки "Продолжить без ассертов" и "Сохранить лог"
