## Bug Reports

В данном разделе представлены баг-репорты, выявленные в ходе регрессионного тестирования iOS-приложения IVI после обновления вертикального плеера, режима PiP, галереи серий, раздела настроек и блока «Продолжить просмотр».

Тестирование проводилось с привязкой к задачам формата IOS-XXXXX и направлено на подтверждение корректности исправлений и новой реализации функционала.

---

## Зафиксированные отклонения

В ходе проверки обнаружены следующие проблемы:

### IOS-17802.2
После завершения бесплатной серии в режиме PiP не открывается экран с предложением оформить подписку.  
(Видео: IMG_1271)

---

### IOS-18599.1 — Раздел настроек плеера

Обнаружены критичные отклонения:

- окно настроек самопроизвольно закрывается (IMG_1273);
- вкладка «Качество» размещена внутри вкладки «Настройки» (IMG_1272);
- при изменении качества видео, находящееся на паузе, автоматически начинает воспроизведение (IMG_1274);
- громкость изменяется самостоятельно без действий пользователя.

---

### IOS-15934.3 — Блок «Избранное» и приоритет разделов

Выявлены расхождения с ожидаемым поведением:

- заголовок раздела отображается как «Моё» вместо ожидаемого;
- отображается блок «Продолжить просмотр» вместо «Смотреть позже»;
- нарушен порядок блоков (Продолжить просмотр → Буду смотреть → Загрузки → Просмотры);
- отображается пустой блок «Покупки».  
(Скриншоты: IMG_1275, IMG_1277)

## Структура баг-репорта

Каждый баг-репорт включает:

- ID задачи (IOS-XXXXX);
- окружение (устройство, версия iOS);
- предусловия;
- шаги воспроизведения;
- ожидаемый результат;
- фактический результат;
- уровень серьёзности (Severity);
- приоритет (Priority);
- прикреплённые скриншоты или видео.
