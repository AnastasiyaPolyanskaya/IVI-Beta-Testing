# Тест-кейсы - Проект №7 (Онбординг и контекстное меню)

В этом разделе размещён файл с тест-кейсами, применяемыми для проверки появления онбординга и работы контекстного меню в iOS-приложении IVI.

## Содержимое

- **21.11.25_Тест-кейсы_ИВИ.xlsx**  
  В тест-кейсах проверяются:
  - отображение онбординга над горизонтальным постером;
  - условия появления онбординга;
  - возможность скрытия долгим нажатием;
  - работа контекстного меню;
  - сценарии с повторным входом;
  - нестандартные состояния (онбординг не появился / появился поздно).

## Применимость

Тест-кейсы относятся к мобильной версии IVI для iOS и охватывают UI/UX-поведение онбординга и взаимодействие пользователя с контекстным меню.
