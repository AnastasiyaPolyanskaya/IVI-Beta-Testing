# Проект №7 - IVI: Онбординг и контекстное меню (iOS)

## Описание

Проведено исследовательское и функциональное тестирование отображения онбординга, обучающего пользователя открытию контекстного меню над горизонтальными постерами на главной странице iOS-приложения IVI.  

Цель тестирования — проверить появление онбординга, возможность его скрытия долгим нажатием и корректную работу контекстного меню.

## Окружение

- Устройство: iPhone 14 Pro Max  
- ОС: iOS 18.6.2  
- Приложение: IVI версия 25.11.3  
- Тип тестирования: Exploratory + UI/UX  
- Применимость: мобильное приложение (iOS)

## Проверенные сценарии

- Появление онбординга при скролле до горизонтального постера  
- Отображение обучающего слоя поверх контента  
- Возможность скрытия онбординга долгим нажатием  
- Открытие контекстного меню над постером  
- Проверка корректного поведения онбординга при повторном входе  
- Связанное поведение элементов (зависимость от появления онбординга)
