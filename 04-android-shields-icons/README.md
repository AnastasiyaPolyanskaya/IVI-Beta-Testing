# Проект №4 - IVI: Тестирование шильдов и иконок в Android-приложении

## Описание
Проведено исследовательское тестирование корректности отображения шильдов, иконок, тегов и служебных меток в Android-приложении IVI.  
Проверка включала отображение промо-ярлыков, возрастных ограничений, категорий контента, статусов просмотров и других графических элементов интерфейса.

## Окружение
- ОС: Android 14  
- Устройства: Android смартфон / эмулятор  
- Приложение: IVI для Android  

## Проверенные сценарии
- отображение шильдов в карточках фильмов  
- отображение меток на плитках контента  
- корректность отображения тегов при скролле  
- проверка состояния элементов при переключении вкладок  
- проверка отображения категорий и возрастных ограничений  
- проверка реакций UI при открытии карточки фильма  
- проверка корректности загрузки изображений (placeholder, blur, качество)

## Найденные дефекты

- некоторые шильды отображаются обрезанными на малых экранах  
- некорректно подгружаются иконки при плохом интернете  
- несоответствие цвета или стиля отдельных меток  

Формальные баг-репорты будут размещены в папке **bug-reports/**.
