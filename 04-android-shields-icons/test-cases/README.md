# Тест-кейсы - Проект №4 (Шильды и иконки в Android-приложении IVI)

В данном разделе размещены тест-кейсы, выполненные в рамках проверки корректности отображения визуальных элементов (шильдов, тегов, иконок) в Android-приложении IVI.

## Основные направления тестирования

Тест-кейсы включают:

- проверку отображения возрастных ограничений;
- проверку стилей и цвета шильдов в карточках фильмов;
- отображение тегов и промо-меток при скролле;
- стабильность отображения при смене вкладок;
- реакцию UI при обновлении данных или загрузке контента;
- проверку корректности размеров и положения иконок.

## Формат тест-кейсов

Каждый тест-кейс содержит:

- **ID**
- **Название**
- **Предусловия**
- **Шаги воспроизведения**
- **Ожидаемый результат**

## Применимость

Тест-кейсы относятся к UI/UX тестированию Android-версии приложения IVI и позволяют проверять стабильность и корректность отображения визуальных элементов.
