# Баг-репорты - Проект №4 (Шильды и иконки в Android-приложении IVI)

В данном разделе размещены баг-репорты, выявленные при исследовательском и функциональном тестировании отображения шильдов, иконок и тегов в Android-приложении IVI.

## Основные направления проверок
Баги относятся к следующим категориям:

- некорректное отображение шильдов в карточках фильмов  
- неправильная отрисовка промо-меток и возрастных ограничений  
- проблемы с отображением иконок при скролле  
- задержки или ошибки загрузки изображений  
- несоответствие цвета или стиля отдельных меток  
- неправильное отображение тегов при переходах между вкладками  

## Содержимое
Файл баг-репортов включает:

- описание найденных дефектов  
- шаги воспроизведения  
- ожидаемый и фактический результаты  
- окружение  
- уровень влияния ошибки на пользователя  

## Применимость
Материалы относятся к UI/UX тестированию Android-приложения IVI и охватывают визуальную проверку элементов, влияющих на пользовательский опыт.
