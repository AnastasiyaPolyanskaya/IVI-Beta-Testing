# Bug Reports — IVI Beta Testing

В этом разделе размещены баг-репорты, найденные в процессе исследовательского и функционального тестирования сервиса IVI.

Баг-репорты оформлены по стандартной структуре: шаги воспроизведения, ожидаемый результат, фактический результат, приоритет и вложения (скриншоты или видео).

## Что содержится в этой папке

- дефекты интерфейса (UI)
- ошибки логики и функциональности
- проблемы воспроизведения контента
- ошибки авторизации и регистрации
- баги в рекомендациях и поиске
- мобильные дефекты (адаптивность, отображение)
- сетевые ошибки и нестабильность API (если наблюдалась)

## Структура баг-репорта

Каждый баг-репорт оформляется по шаблону:

- **ID:** уникальный номер дефекта  
- **Название:** краткое описание ошибки  
- **Окружение:** устройство, ОС, браузер/версия приложения  
- **Предусловия:** если есть  
- **Шаги воспроизведения:** детально  
- **Ожидаемый результат:** что должно быть  
- **Фактический результат:** что происходит на самом деле  
- **Приоритет:** Low / Medium / High / Critical  
- **Вложения:** скриншоты, видео

## Назначение раздела

Раздел помогает систематизировать найденные дефекты, отслеживать качество работы сервиса IVI и демонстрировать навыки тестирования в реальном продукте.
