# Проект №6 - IVI: Детский профиль и защита контента 18+ (iOS)

## Описание

Проведено исследовательское и функциональное тестирование раздела «Детский профиль» и механизма защиты контента 18+ в iOS-приложении IVI.  
Цель — проверить корректность создания детского аккаунта, работу выбора возраста/месяца рождения, а также проверку PIN-кода при попытке открыть контент 18+.

## Окружение

- Устройство: iPhone 14 Pro  
- ОС: iOS 18.6.2  
- Приложение: IVI, версия 25.11.2.0  
- Тип тестирования: Exploratory + Functional  
- Применимость: iOS-мобильная версия

## Проверенные сценарии

- Создание детского профиля  
- Выбор месяца рождения ребёнка  
- Выбор возраста ребёнка  
- Проверка виброотклика при настройке  
- Настройка PIN-кода для контента 18+  
- Проверка запуска контента 18+ во взрослом профиле  
- Проверка появления экрана PIN-защиты  
- Проверка корректности ограничения доступа

## Найденные дефекты

1. **Вибрация при выборе месяца и возраста ребёнка в детском профиле**  
   Детали — в папке `bug-reports/`.

2. **Фильм с маркировкой 18+ запускается без ввода PIN-кода**  
   Критический дефект безопасности.  
   Формальный баг — `ID: ив1.4` (см. баг-репорты).
