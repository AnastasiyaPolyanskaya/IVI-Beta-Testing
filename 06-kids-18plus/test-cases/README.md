# Тест-кейсы - Проект №6 (Детский профиль и защита 18+)

В данном разделе находятся тест-кейсы, подготовленные для проверки работы функционала детского профиля и ограничения контента 18+ на платформе iOS.

## Содержимое

- **14.11.25_Тест-кейсы_ИВИ.xlsx**  
  Файл включает тест-кейсы по следующим направлениям:
  - создание детского профиля;
  - выбор месяца рождения ребёнка;
  - выбор возраста ребёнка;
  - настройка PIN-кода 18+;
  - проверка доступа к контенту 18+;
  - работа экрана PIN-защиты;
  - проверка ошибок и нестандартных сценариев.

## Применимость

Тест-кейсы относятся к iOS-версии приложения IVI и охватывают функциональное тестирование и проверку пользовательского опыта.
