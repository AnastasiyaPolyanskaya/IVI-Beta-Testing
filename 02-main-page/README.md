# Проект №2 - IVI: Тестирование главной страницы (Web)

## Описание
Проведено исследовательское и функциональное тестирование главной страницы сервиса IVI в веб-версии.  
Особое внимание уделено корректности отображения контента, работе баннеров, карточек фильмов, навигации и пользовательских сценариев взаимодействия с основными элементами.

## Окружение
- ОС: Windows 11  
- Браузеры: Chrome, Яндекс.Браузер  
- Платформа: Web (десктоп)

## Проверенные сценарии
- Отображение рекламных и промо-баннеров.  
- Переходы по основным разделам меню.  
- Открытие карточек фильмов с главной страницы.  
- Проверка корректности подгрузки контента при скролле.  
- Проверка кликабельности кнопок и ссылок.  
- Проверка корректности отображения обложек, названий и рейтингов.  
- Проверка работы всплывающих подсказок и попапов (если присутствуют).
