# Баг-репорты - Проект №2 (Главная страница IVI)

В данном разделе представлены баг-репорты, выявленные при исследовательском и функциональном тестировании главной страницы сервиса IVI (Web).

## Основные группы проверок
Баги относятся к следующим направлениям:

- отображение баннеров и рекламных блоков  
- работа навигационного меню  
- открытие карточек фильмов  
- корректность подгрузки контента при скролле  
- отображение рейтингов, обложек и описаний  
- кликабельность элементов интерфейса  

## Содержимое
Файл баг-репортов содержит:

- описание выявленных дефектов  
- шаги воспроизведения  
- фактические и ожидаемые результаты  
- окружение и степень влияния на пользователя  

## Применимость
Материалы используются для анализа стабильности Web-версии главной страницы IVI.
