# Тест-кейсы - Проект №2 (Главная страница IVI)

В данном разделе размещены тест-кейсы, составленные для проверки работы главной страницы веб-версии IVI.

## Содержимое
Файл тест-кейсов содержит сценарии для проверки следующих элементов и функций:

- отображение баннеров и промо-блоков  
- корректность работы навигационного меню  
- отображение карточек фильмов  
- переходы в карточку фильма  
- корректность подгрузки контента при скролле  
- кликабельность кнопок и ссылок  
- проверка корректности отображения обложек, рейтингов и названий  

## Применимость
Тест-кейсы относятся к веб-версии IVI и предназначены для функционального и UI/UX тестирования главной страницы.
