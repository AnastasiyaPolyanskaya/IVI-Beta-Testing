# Баг-репорты - Smart TV версия IVI

В данном разделе собраны дефекты, выявленные при тестировании приложения IVI на Smart TV (LG WebOS / Samsung Tizen).

## Основные направления найденных ошибок

В ходе тестирования были зафиксированы следующие группы проблем:

- **Ошибки навигации**  
  Фокус «теряется», перескакивает на несуществующие элементы или некорректно возвращается назад.

- **UI-несоответствия**  
  Неверное отображение карточек, обрезанные тексты, неправильное выделение активного элемента.

- **Проблемы в плеере**  
  Некорректный переход в полноэкранный режим, зависания при перемотке, неверная работа кнопок паузы/возврата.

- **Задержки и фризы**  
  Приложение тормозит при быстрой навигации или при загрузке больших каруселей.

- **Ошибки взаимодействия с пультом**  
  Двойное срабатывание кнопки OK, неработающая кнопка Back в отдельных сценариях.
