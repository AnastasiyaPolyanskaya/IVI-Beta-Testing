# Тест-кейсы - Smart TV версия IVI

В разделе представлены тест-кейсы, подготовленные в рамках тестирования версии IVI для Smart TV (LG WebOS / Samsung Tizen).

## Что проверяется в тест-кейсах

- корректность навигации с пульта (стрелки, OK, Back);  
- отображение главной страницы и карточек контента;  
- открытие фильма/сериала и запуск воспроизведения;  
- работа мини-плеера и полноэкранного режима;  
- поведение интерфейса при быстром перемещении фокуса;  
- стабильность UI и плеера.

## Содержание

- **TC-001 — Навигация по главной странице**
- **TC-002 — Открытие карточки фильма**
- **TC-003 — Запуск воспроизведения**
- **TC-004 — Работа мини-плеера**
- **TC-005 — Проверка возвращения назад**
- **TC-006 — Перемещение фокуса в каруселях**
- **TC-007 — Быстрая навигация (стресс-тест пульта)**
