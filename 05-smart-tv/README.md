# Проект №5 - IVI: Тестирование Smart TV версии

## Описание

В рамках данного проекта была проведена проверка основных пользовательских сценариев в Smart TV версии IVI.  
Особое внимание уделено навигации с помощью пульта, корректности отображения UI-элементов на большом экране, работе карточек фильмов, лент рекомендаций и поведению фокуса.

Цель тестирования — определить проблемы интерфейса, влияющие на удобство взаимодействия с приложением на телевизорах.

## Окружение

- **Устройства:** LG Smart TV, Samsung Smart TV (WebOS / Tizen)  
- **Приложение:** IVI Smart TV  
- **Тип управления:** стандартный пульт (стрелки, OK, Back)  

## Тип тестирования

- Exploratory testing  
- UI/UX testing  
- Navigation testing  
- Display layout testing  

## Проверенные сценарии

- запуск приложения и загрузка главной страницы;
- первая навигация по каталогу с помощью стрелок пульта;
- корректность перемещения фокуса между элементами;
- поведение фокуса на каруселях (листы, горизонтальная и вертикальная навигация);
- открытие карточек фильмов;
- работа кнопок: *Play*, *Trailer*, *Добавить в избранное*;
- навигация назад через *Back*;
- воспроизведение контента (мини-плеер + полноэкранный режим);
- корректность отображения элементов на 4K / Full HD;
- стабильность работы и отсутствие подвисаний при быстром скролле.



Формальные баг-репорты находятся в папке `/bug-reports/`.

## Состав проекта

- `/test-cases/` — тест-кейсы по Smart TV версии  
- `/bug-reports/` — баг-репорты  
- `/screenshots/` — подтверждающие изображения
