# Тест-кейсы - Проект №9  
IVI: Регрессионное тестирование iOS (Live Activity, Dynamic Island, загрузки)

В данном разделе размещены тест-кейсы, использованные для регрессионного тестирования iOS-приложения IVI в рамках бета-тестирования.

## Описание

Тест-кейсы основаны на официальных инструкциях для проверки iOS-функционала и направлены на валидацию стабильности приложения после обновлений.

Проверяется корректность:
- загрузки сериалов и эпизодов;
- работы Live Activity;
- отображения прогресса загрузки в Dynamic Island;
- поведения приложения при сворачивании и блокировке устройства;
- основных пользовательских экранов.

## Покрываемый функционал

### Загрузка контента
- Загрузка целого сезона
- Остановка загрузки отдельных серий
- Продолжение загрузки остальных эпизодов
- Корректное завершение загрузки

### Live Activity и Dynamic Island
- Отображение мини-превью загрузки в Dynamic Island
- Обновление прогресса загрузки
- Отображение виджета Live Activity на экране блокировки
- Переход на экран «Загрузки» по нажатию на виджет

### Основные экраны
- Продолжить просмотр
- Карточка персоны
- Страница покупок
- Страница с чеками
- Страница просмотров
- Страница ТВ-трансляции

## Типы тестирования
- Регрессионное
- Функциональное
- UI/UX

## Формат тест-кейсов

Тест-кейсы включают:
- предусловия;
- пошаговые сценарии;
- ожидаемые результаты;
- фактические результаты (при отклонениях);
- статус выполнения.

## Примечание

Тест-кейсы предназначены для проверки стабильности приложения IVI на iOS-устройствах и могут быть повторно использованы при последующих релизах.
