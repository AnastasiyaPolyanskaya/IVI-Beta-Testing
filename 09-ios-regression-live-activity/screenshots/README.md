# Screenshots

В данном разделе собраны скриншоты, полученные в ходе тестирования
мобильного приложения **IVI (iOS)**.

Скриншоты используются как подтверждающие артефакты выполнения
тест-кейсов и демонстрируют фактическое поведение приложения.

## Что представлено на скриншотах
- экран загрузки сериального контента
- управление загрузкой эпизодов
- отображение прогресса загрузки в Dynamic Island
- Live Activity на экране блокировки
- экран успешного формирования рекомендаций

## Назначение
Скриншоты не являются баг-репортами,  
а используются для:
- визуального подтверждения шагов тест-кейсов
- фиксации корректной работы функциональности
- дополнения тестовой документации проекта
