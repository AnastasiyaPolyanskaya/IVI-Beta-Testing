# Проект №9: IVI — iOS UI Testing (Main Screen & Download)

## Описание
Проект содержит набор тест-кейсов для мобильного приложения **IVI (iOS)**.  
В рамках проекта выполнено **функциональное и UI-тестирование** главной страницы приложения, сценариев загрузки сериального контента, а также системных iOS-интеграций (**Dynamic Island**, **Live Activity**).

Проект оформлен в формате QA-портфолио с тестовой документацией и визуальными артефактами (скриншоты).

## Цель тестирования
- Проверить корректность управления загрузкой эпизодов при скачивании сезона
- Проверить отображение прогресса загрузки в **Dynamic Island**
- Проверить отображение статуса и завершения загрузки через **Live Activity** на экране блокировки
- Проверить сценарий формирования персональных рекомендаций через профиль пользователя

## Тип тестирования
- Functional testing
- UI testing
- Проверка iOS-интеграций (Dynamic Island, Live Activity)

## Окружение
- Устройство: iPhone 14 Pro Max  
- ОС: iOS 18.6.2  

## Тестовые данные
- Пользователь с активной подпиской
- Сериальный контент, доступный для загрузки (более 8 эпизодов в сезоне)

## Объект тестирования
- Главная страница приложения IVI
- Загрузка сериального контента (управление эпизодами)
- Dynamic Island (отображение прогресса загрузки)
- Live Activity на экране блокировки
- Профиль пользователя (формирование рекомендаций)
