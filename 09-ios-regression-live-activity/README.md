# Проект №9 - IVI: Регрессионное тестирование iOS (загрузка контента, Live Activity, Dynamic Island)

## Описание

Проведено регрессионное тестирование iOS-приложения IVI по инструкциям бета-тестирования.  
Проверялась корректность работы загрузки контента, отображение Live Activity на экране блокировки и в Dynamic Island, а также стабильность основных экранов приложения.

Цель тестирования — убедиться, что ключевые пользовательские сценарии работают корректно после обновлений и изменений в приложении.

## Окружение

- Платформа: iOS  
- Устройство: iPhone (с поддержкой Dynamic Island)  
- ОС: iOS  
- Приложение: IVI (бета-версия)  
- Тип тестирования: Регрессионное  
- Канал распространения: Beta / TestFlight  
- Источник сценариев: инструкции бета-тестирования IVI

## Проверяемый функционал

### Загрузка сериалов
- Загрузка целого сезона
- Остановка загрузки отдельных серий
- Корректное продолжение загрузки остальных эпизодов
- Стабильность приложения во время загрузки

### Live Activity и Dynamic Island
- Отображение прогресса загрузки в Dynamic Island
- Отображение Live Activity на экране блокировки
- Обновление статуса загрузки
- Переход на экран «Загрузки» по нажатию на виджет

### Основные экраны приложения
- Продолжить просмотр
- Карточка персоны
- Страница покупок
- Страница с чеками
- Страница просмотров
- Страница ТВ-трансляции

### Подписки и оплата
- Отображение активной подписки
- Корректность UI элементов (иконки, кнопки, плейсхолдеры)

## Артефакты тестирования

- Тест-кейсы
- Чек-листы регрессионного тестирования
- Баг-репорты (при обнаружении дефектов)
- Скриншоты проблемных состояний

## Итог

Регрессионное тестирование позволило проверить стабильность ключевых функций iOS-приложения IVI, связанных с загрузкой контента и системными iOS-возможностями (Live Activity, Dynamic Island).  
Проект демонстрирует опыт работы с реальными тестовыми инструкциями, регрессом и проверкой системных интеграций iOS.
