# Тест-кейсы - Проект №8 (Промокоды и терминология)

В этом разделе размещён файл с тест-кейсами, применяемыми для проверки
корректности текстов интерфейса в сценариях, связанных с промокодами
в iOS-приложении IVI.

## Содержимое

- **Тест-кейсы_ИВИ8.xlsx**

В тест-кейсах проверяются:
- отображение терминологии на экране «О сервисе»;
- корректность текста на экране «Подписки»;
- формулировки, связанные с вводом и активацией промокода;
- соответствие терминов текущему пользовательскому сценарию;
- отсутствие устаревших формулировок, связанных с сертификатами.

## Применимость

Тест-кейсы относятся к мобильной версии IVI для iOS и охватывают
проверку UI-текстов и UX-восприятия интерфейса в сценариях использования промокодов.
