# Скриншоты - Проект №8 (Промокоды и терминология)

В данном разделе размещены скриншоты, подтверждающие выявленные дефекты,
связанные с некорректной терминологией в пользовательских сценариях промокодов
iOS-приложения IVI.

## Содержимое

- **БР_ивпркд9.jpg** — экран «О сервисе», используется термин «сертификат»
- **БР_ивпркд10.jpg** — экран «Подписки», формулировка «Есть сертификат или промокод»
- **БР_ивпркд11.jpg** — экран «Активация промокода», текст «Правила использования сертификатов»

## Назначение

Скриншоты используются как визуальное подтверждение баг-репортов
и демонстрируют несоответствие терминологии текущему пользовательскому сценарию промокодов.
