# Баг-репорты - Проект №8 (Промокоды и терминология)

В данном разделе размещены баг-репорты, выявленные в ходе тестирования
пользовательских экранов iOS-приложения IVI, связанных с использованием промокодов.

## Содержимое

- **Баг-репортыИВИ8.docx**

В баг-репортах зафиксированы следующие дефекты:
- использование устаревшего термина «сертификат» на экране «О сервисе»;
- некорректная формулировка в разделе «Подписки»;
- отображение текста «Правила использования сертификатов» на экране активации промокода;
- несоответствие терминологии текущему пользовательскому сценарию промокодов.

## Применимость

Баг-репорты относятся к мобильной версии IVI для iOS и описывают дефекты UI-текста,
которые могут вводить пользователя в заблуждение при работе с промокодами.
