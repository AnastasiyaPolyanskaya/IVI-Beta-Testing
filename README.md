# IVI Beta Testing

Практический проект по исследовательскому и функциональному тестированию сервиса IVI.  
Репозиторий содержит результаты тестирования различных частей сервиса на платформах Web, iOS, Android и Smart TV.

---

## Содержание репозитория

В проект входят следующие направления тестирования:

1. **Форма оплаты (Web + Mobile)**  
   Исследовательское тестирование страницы оформления подписки и реакции UI на ввод данных карты.

2. **Главная страница IVI (Web)**  
   Проверка баннеров, карточек фильмов, навигации, переходов, корректности отображения элементов.

3. **Промокоды, подписки и всплывающие окна (Web + Mobile)**  
   Проверка ввода промокодов, поведения подписки, отображения информационных окон.

4. **Шильды и иконки в Android-приложении**  
   Проверка корректного отображения шильдов, тегов и меток в каталоге и карточках фильмов.

5. **Версия приложения для Smart TV**  
   Проверка навигации с пульта, фокусировки элементов, логики переходов и отображения контента.

6. **Онбординг и контекстные меню (iOS / Android)**  
   Проверка первых экранов, обучающих подсказок, контекстных меню, жестов и поведения элементов.

## Цель проекта

Провести анализ стабильности и корректности работы ключевых пользовательских сценариев сервиса IVI на разных устройствах и платформах.  
Выявить функциональные дефекты, несоответствия UI, проблемы навигации и поведения элементов.

---

## Типы тестирования

- исследовательское (Exploratory)  
- функциональное  
- регрессионное  
- UI/UX-проверки  
- кроссплатформенное тестирование  
- мобильное тестирование
