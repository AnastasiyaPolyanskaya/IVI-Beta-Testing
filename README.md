# IVI Beta Testing

Практический проект по исследовательскому и функциональному тестированию сервиса IVI в рамках открытого бета-тестирования.

В проекте представлены:
- тест-кейсы  
- чек-листы  
- сценарии тестирования  
- баг-репорты  
- скриншоты и подтверждения ошибок  
- результаты анализа работы сервиса  

---

## Цель проекта

Проверить стабильность ключевых функций сервиса IVI, выявить UI-несоответствия, ошибки навигации, проблемы с воспроизведением контента и другие дефекты, влияющие на пользовательский опыт.

---

## Структура проекта

- **test-cases/** — тест-кейсы по функционалу сервиса  
- **checklists/** — проверочные списки для регресса  
- **bug-reports/** — оформленные баг-репорты  
- **screenshots/** — скриншоты ошибок и подтверждающие материалы  
- **devtools/** — HAR-файлы, сетевые логи (если применимо)

---

## Инструменты

- Веб-версия IVI  
- Мобильное приложение (iOS / Android)  
- Chrome DevTools  
- Charles Proxy / HAR-файлы  

---

## Результаты проекта

Проект показывает навыки:
- анализа требований  
- исследовательского и функционального тестирования  
- составления тест-кейсов и сценариев  
- оформления баг-репортов  
- работы с DevTools и сетевыми логами  
- систематизации тестовой документации
