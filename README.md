# IVI Beta Testing

Практический проект по исследовательскому, функциональному и UI/UX-тестированию частей сервиса IVI на платформах Web, iOS, Android и Smart TV.

Репозиторий содержит тест-кейсы, баг-репорты, сценарии тестирования и скриншоты.  
Материалы структурированы по подпроектам.

## Структура репозитория

### 01-payment-form — Тестирование формы оплаты (Web + Mobile)
Исследовательское тестирование страницы оформления подписки: отображение UI, реакция формы на ввод реквизитов разных банков, обновление визуального состояния карты.  
Содержимое: тест-кейсы, баг-репорты, скриншоты.

### 02-main-page — Главная страница (Web + Mobile)
Проверка загрузки контента, стабильности интерфейса, поведения страницы при включенном VPN, корректности работы плеера.  
Содержимое: тест-кейсы, баг-репорты.

### 03-promocodes-subscriptions — Промокоды и подписки (iOS)
Тестирование сценариев активации промокодов, работы раздела «Смотреть позже», корректности отображения элементов интерфейса.  
Содержимое: тест-кейсы, баг-репорты.

### 04-android-shields-icons — Визуальные элементы (Android)
UI-проверка шильдов, иконок подписок и корректности адаптации графики на Android-устройствах.  
Содержимое: тест-кейсы, результаты проверки.

### 05-smart-tv — Smart TV тестирование
Проверка воспроизведения контента, стабильности плеера, реакции интерфейса в Smart TV (webOS).  
Содержимое: тест-кейсы, баг-репорты.

### 06-kids-18plus — Детский профиль и защита 18+ (iOS)
Функциональное тестирование: создание детского профиля, выбор возраста, работа PIN-кода, доступ к контенту 18+.  
Содержимое: тест-кейсы, баг-репорты.

### 07-onboarding-context-menu — Онбординг и контекстное меню (iOS)
Проверка отображения онбординга, сценария долгого нажатия, корректной работы контекстного меню над постерами.  
Содержимое: тест-кейсы, баг-репорты.

## Цель репозитория
Демонстрация навыков ручного тестирования:
- составление тест-кейсов;
- оформление баг-репортов;
- исследовательское тестирование;
- тестирование Web, iOS, Android, Smart TV;
- структурирование артефактов тестирования.
