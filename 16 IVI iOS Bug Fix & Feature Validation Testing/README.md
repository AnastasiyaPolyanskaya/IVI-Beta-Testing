# Project 16  IVI iOS Functional Testing

Проект посвящён тестированию мобильного приложения **IVI** после обновлений функциональности и исправления дефектов в iOS версии приложения.

В рамках тестирования проверялась корректность исправленных ошибок, стабильность работы приложения, а также новый функционал воспроизведения контента на заблокированном экране.

---

# Цель тестирования

Подтвердить корректность работы исправленных дефектов и нового функционала приложения, а также убедиться в отсутствии регрессии в ключевых пользовательских сценариях.

---

# Проверяемые задачи

В ходе тестирования были проверены следующие изменения.

## 1. Исправление краша при вводе текста свайпом

Ранее приложение падало при вводе текста свайпом в поле авторизации или регистрации на устройствах с iOS 16–17.

### Проверка
- открытие флоу регистрации;
- ввод текста свайпом.

### Ожидаемый результат
Текст корректно вводится в поле без падения приложения.

---

## 2. Исправление очистки поля поиска

Ранее поле поиска не очищалось при нажатии на кнопку очистки.

### Проверка
- открыть поиск;
- ввести любой текст;
- нажать на крестик в поле поиска.

### Ожидаемый результат
Поле поиска полностью очищается.

---

## 3. Обновление раздела «Буду смотреть»

Раздел был переведён на новый внутренний сервис.  

### Проверки
- добавление контента в раздел;
- отображение статуса добавления на карточке контента;
- удаление контента из раздела;
- корректное отображение списка контента.

### Ожидаемый результат
Раздел работает корректно, все действия выполняются без ошибок.

---

## 4. Воспроизведение на заблокированном экране

Добавлен новый функционал воспроизведения контента при заблокированном экране устройства.

### Проверяемые плееры
- ТВ-плеер
- обычный плеер
- плеер мини-драм
- режим «Картинка в картинке»

---

### Кейс 1 — пользователь с активной подпиской

После блокировки экрана возможны два корректных сценария:

- контент продолжает воспроизводиться;
- контент ставится на паузу.

В обоих случаях отображается виджет управления воспроизведением на заблокированном экране.

---

### Кейс 2 — пользователь без подписки

После блокировки экрана воспроизведение контента прекращается.

---

# Окружение тестирования

- Платформа: **iOS**
- Версии iOS: **16 – 17**
- Приложение: **IVI**
