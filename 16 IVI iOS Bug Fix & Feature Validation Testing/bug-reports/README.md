## Bug Reports

В данной папке представлены баг-репорты, обнаруженные в процессе тестирования мобильного приложения **IVI** для платформы **iOS**.

Тестирование проводилось после внедрения нового функционала воспроизведения контента на заблокированном экране устройства. В ходе проверки были выявлены дефекты, связанные с некорректной работой воспроизведения и системного виджета управления плеером. :contentReference[oaicite:0]{index=0}

---

## Обнаруженные дефекты

В ходе тестирования были выявлены следующие проблемы:

- отсутствие воспроизведения ТВ-каналов на заблокированном экране;
- отсутствие системного виджета управления воспроизведением;
- прекращение воспроизведения мини-драм после блокировки устройства;
- некорректная работа виджета управления воспроизведением при использовании режима **Picture-in-Picture (PiP)**.

Все дефекты зафиксированы в баг-репортах с указанием шагов воспроизведения, ожидаемого и фактического результата.

---

## Назначение баг-репортов

Баг-репорты используются для:

- фиксации обнаруженных дефектов;
- документирования фактического поведения приложения;
- передачи информации команде разработки;
- подтверждения найденных проблем с помощью скриншотов и видео.

---

## Окружение тестирования

- Устройство: **iPhone 14 Pro Max**
- Платформа: **iOS 18.6.2**
- Приложение: **IVI**
- Версия приложения: **26.3.3**
