## Test Cases

В данной папке представлены тест-кейсы, разработанные для проверки исправленных дефектов и нового функционала мобильного приложения **IVI** для платформы **iOS**.

Тест-кейсы охватывают ключевые пользовательские сценарии и направлены на подтверждение корректной работы обновлённого функционала приложения, а также проверку отсутствия регрессии после внесённых изменений.

---

## Покрываемые области тестирования

### Ввод текста в поле авторизации/регистрации

Проверяется исправление дефекта, при котором приложение падало при вводе текста свайпом.

Проверяемый сценарий:
- открытие формы регистрации;
- ввод текста в поле ввода с использованием свайпа.

Ожидаемый результат — текст корректно вводится в поле без падения приложения.

---

### Работа поля поиска

Проверяется корректность работы кнопки очистки поисковой строки.

Проверяемый сценарий:
- открытие раздела поиска;
- ввод любого запроса;
- нажатие на кнопку очистки (крестик).

Ожидаемый результат — поисковая строка очищается.

---

### Работа раздела «Буду смотреть»

Проверяется корректность работы раздела после перехода на новый внутренний сервис.

Проверяемые сценарии:

- добавление контента в раздел;
- корректное отображение статуса добавления на карточке контента;
- удаление контента из раздела;
- отображение списка контента в разделе.

---

### Воспроизведение контента на заблокированном экране

Проверяется новый функционал воспроизведения контента при заблокированном экране устройства.

Тестирование проводится для различных типов плееров:

- ТВ-плеер  
- обычный видеоплеер  
- плеер мини-драм  
- режим **«Картинка в картинке» (PiP)**  

Также проверяются сценарии для пользователей:

- с активной подпиской;
- без активной подписки.

---

## Структура тест-кейсов

Каждый тест-кейс содержит следующие элементы:

- **ID тест-кейса**
- **Название тест-кейса**
- **Предусловия**
- **Шаги воспроизведения**
- **Ожидаемый результат**
- **Фактический результат**
- **Окружение**
- **Статус выполнения теста**

---

- Устройство: **iPhone 14 Pro Max**
- Платформа: **iOS 18.6.2**
- Приложение: **IVI**
