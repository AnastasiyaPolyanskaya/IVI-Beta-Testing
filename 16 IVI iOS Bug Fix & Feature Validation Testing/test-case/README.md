## Test Cases

В данной папке представлены тест-кейсы, разработанные для проверки исправленных дефектов и нового функционала мобильного приложения IVI для iOS.

Тест-кейсы направлены на проверку стабильности приложения, корректности работы пользовательских сценариев и подтверждение исправления ранее обнаруженных ошибок.

---

## Проверяемые области

В рамках тестирования были подготовлены тест-кейсы для следующих функциональных областей приложения:

### Исправление краша при вводе текста
Проверяется отсутствие краша приложения при вводе текста свайпом в поле авторизации или регистрации на устройствах с iOS 16–17.

### Поле поиска
Проверяется корректная работа кнопки очистки поля поиска при нажатии на крестик.

### Раздел «Буду смотреть»
Проверяется корректность работы раздела после перехода на новый внутренний сервис:
- добавление контента в раздел;
- корректное отображение статуса добавления на карточке контента;
- удаление контента из раздела;
- отображение списка контента.

### Воспроизведение на заблокированном экране
Проверяется новый функционал воспроизведения контента при заблокированном экране устройства для различных типов плееров:
- ТВ-плеер;
- обычный плеер;
- плеер мини-драм;
- режим «Картинка в картинке».

Тестирование проводится для пользователей:
- с активной подпиской;
- без активной подписки.

---

## Структура тест-кейсов

Каждый тест-кейс содержит следующие поля:

- ID тест-кейса  
- название тест-кейса  
- предусловия  
- шаги воспроизведения  
- ожидаемый результат  
- фактический результат  
- окружение тестирования  
- результат выполнения теста

---

## Окружение тестирования

- Устройства: iPhone  
- Версия платформы: iOS 16–17  
- Приложение: IVI
