## Test Cases

В данной папке представлены тест-кейсы для проверки пользовательского интерфейса и функциональности главной страницы приложения IVI (версия 26.3.1).

Тестирование направлено на подтверждение корректной работы настроек плеера, поведения блока «Продолжить просмотр», а также проверки визуальных изменений интерфейса приложения.

---

## Покрываемые сценарии

### 1. Проверка настроек плеера

В рамках тестирования проверяется корректное отображение и работа элементов настроек плеера:

- отображение параметра «Качество»;
- отображение текущего режима качества (Auto);
- отображение параметра «Скорость воспроизведения»;
- отображение текущего значения скорости воспроизведения;
- отображение переключателя «Режим Марафона»;
- корректная работа переключателя режима марафона;
- отображение параметра разрешения вертикального режима.

---

### 2. Проверка блока «Продолжить просмотр»

Проверяются пользовательские сценарии взаимодействия с блоком:

- появление контента в блоке после начала просмотра;
- удаление контента из блока после завершения просмотра контента.

---

### 3. Проверка визуальных изменений интерфейса

Дополнительно проверяются обновления визуальных элементов приложения:

- обновление иконки приложения;
- обновление splash-screen при запуске приложения.

---

## Типы тестирования

В рамках тест-кейсов применяются следующие виды тестирования:

- UI Testing  
- Functional Testing  
- Regression Testing  

---

## Окружение тестирования

- Устройство: iPhone 14 Pro Max  
- Версия iOS: 18.6.2 – 18.6.3  
- Приложение: IVI  
- Версия приложения: 26.3.1  

---

## Результаты выполнения

Большинство тест-кейсов выполнены со статусом **Passed**, что подтверждает корректную работу настроек плеера и пользовательских сценариев.

В ходе тестирования выявлены дефекты, связанные с визуальными элементами приложения (иконка приложения и splash-screen), которые зафиксированы в тест-кейсах со статусом **Failed**.
