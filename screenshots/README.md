# Screenshots — IVI Beta Testing

В этом разделе размещаются визуальные материалы, использованные в процессе исследовательского и функционального тестирования сервиса IVI.

Скриншоты помогают фиксировать найденные дефекты, подтверждать шаги воспроизведения и иллюстрировать поведение интерфейса в различных сценариях.

## Что содержится в этой папке

- скриншоты и подтверждения ошибок  
- визуализация шагов воспроизведения  
- ошибки отображения и верстки  
- примеры некорректной работы функционала  
- поддерживающие материалы к баг-репортам  
- материалы для анализа поведения сервиса  

## Назначение раздела

Раздел предназначен для хранения визуальных доказательств найденных дефектов, а также материалов, которые используются для составления баг-репортов и анализа качества приложения.
