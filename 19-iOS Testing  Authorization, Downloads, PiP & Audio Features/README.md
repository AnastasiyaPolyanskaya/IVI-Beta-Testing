Практический проект по ручному тестированию мобильного приложения **IVI** для платформы **iOS**.

Проект направлен на проверку исправленных дефектов, стабильности приложения и нового функционала, включая авторизацию, загрузку контента, работу плеера и новый раздел «Аудио».

---

## Цель проекта

Проверить корректность работы исправленных ошибок, нового функционала и убедиться в отсутствии регрессии в ключевых пользовательских сценариях приложения.

---

## Объект тестирования

Мобильное приложение **IVI**

- Платформа: **iOS**
- Тип тестирования: **Manual QA**

---

## Проверяемые функциональные области

### 1. Авторизация

Проверено исправление краша на новом экране авторизации.

Проверяемые сценарии:
- вход по коду авторизации;
- вход при активации сертификата из неавторизованного состояния.

Ожидаемый результат — вход выполняется корректно, приложение не падает.

---

### 2. Загрузка контента

Проверено исправление ошибок, связанных с загрузкой.

Проверяемые сценарии:
- постановка контента на скачивание;
- пауза и возобновление загрузки;
- загрузка в фоновом режиме.

Ожидаемый результат — загрузка выполняется корректно, краши отсутствуют.

---

### 3. Воспроизведение скачанного контента

Проверено исправление ошибки, при которой частично просмотренный контент запускался с начала.

Проверяемый сценарий:
- запуск скачанного контента после частичного просмотра.

Ожидаемый результат — воспроизведение начинается с последней точки просмотра.

---

### 4. Работа режима Picture-in-Picture (PiP)

Проверено исправление работы PiP после переключения ТВ-каналов.

Проверяемый сценарий:
- переход между каналами через карусель;
- активация режима PiP.

Ожидаемый результат — корректный переход в режим Picture-in-Picture.

---

### 5. Раздел «Аудио»

Проверен новый функционал приложения.

Проверяемые сценарии:
- отображение раздела в тапбаре;
- отображение подборок (включая ТОП-10);
- корректность отображения карточек контента;
- запуск аудиоконтента;
- работа элементов плеера.

Ожидаемый результат — раздел отображается корректно, визуальные дефекты отсутствуют, плеер работает стабильно.
