## Скриншоты — Проект № 12 (Обновлённые интерфейсные компоненты IVI)

В данном разделе размещены скриншоты, подтверждающие выявленные дефекты в обновлённых интерфейсных компонентах мобильного приложения IVI (Android и iOS). Материалы иллюстрируют проблемы с отображением текста, локализацией и функциональностью ключевых экранов.

## Содержимое

- **Экран раздела «Промокоды»** — демонстрирует сообщение с устаревшей терминологией («Активация сертификата…») и ссылкой на веб‑сайт.
- **Экран загрузки контента** — показывает несоответствие между выбранным языком интерфейса и обозначением размера файла (например, «MB» вместо локализованного варианта).
- **Экран установки ПИН‑кода** — фиксирует некорректное отображение сообщений об ошибках (например, нелокализованный текст ошибки).
- **Профиль пользователя (офлайн‑режим)** — иллюстрирует неправильное состояние кнопки выхода из аккаунта при отключённом интернете.
- **Чат поддержки** — отображает сбой при открытии диалога после авторизации через «Сообщить о проблеме».

## Назначение

Скриншоты выполняют следующие функции:

- служат визуальным подтверждением описанных в баг‑репортах дефектов;
- демонстрируют контекст возникновения ошибок (конкретные экраны и сценарии);
- помогают разработчикам и дизайнерам точно локализовать проблемы с отображением элементов;
- фиксируют несоответствия в локализации и стилистике интерфейса;
- используются в качестве доказательной базы при обсуждении исправлений с командой.

## Применимость

Материалы относятся к:

- актуальным версиям приложения IVI для Android (ОС 14+) и iOS (ОС 16+);
- ключевым сценариям: активация промокодов, загрузка контента, настройка безопасности (ПИН‑код), управление аккаунтом, взаимодействие с поддержкой.

## Примечания

- Все скриншоты сделаны на реальных устройствах с зафиксированными настройками (язык, регион, сетевое состояние).
- Для каждого дефекта приложены снимки до и после воспроизведения (где применимо).
- Файлы сохранены в формате PNG с высоким разрешением для чёткой визуализации элементов интерфейса.
