## Проект №12 — IVI: Тестирование обновлённых интерфейсных компонентов и логики взаимодействия

### Описание

Проведено комплексное тестирование обновлённых элементов интерфейса и ключевых сценариев взаимодействия в приложении IVI. Проверка охватила:

- поля ввода текста и логику их работы;
- механизм установки ПИН‑кода;
- отображение метаданных (в т. ч. размера файла) с учётом системного языка;
- доступность функциональных кнопок при отсутствии сети;
- сценарии работы чата поддержки.

Цель — убедиться, что обновления не нарушили базовую функциональность и соответствуют ожидаемому поведению.

### Окружение

- **ОС:** (iOS 18.6.2) ;
- **Устройства:** iPhone 14 Pro ;
- **Приложение:** 26.1.3 версия IVI для мобильной платформы;
- **Сетевые условия:** онлайн и офлайн‑режимы;
- **Системные настройки:** различные языки интерфейса (включая нерусские).

### Проверенные сценарии

1.  **Поля ввода текста**
    - поиск;
    - редактирование имени, email, телефона в профиле;
    - активация промокода/сертификата;
    - регистрация и авторизация;
    - ввод email/телефона в чате шеринга;
    - ввод email для получения чеков (у пользователя без email).
2.  **Установка ПИН‑кода**
    - переход в настройки профиля;
    - активация опции «Пин‑код — Добавить»;
    - переключение свитчера;
    - ввод и подтверждение ПИН‑кода.
3.  **Отображение размера файла**
    - загрузка контента в КК «Остановка на лето»;
    - смена языка в настройках качества видео;
    - проверка написания размера файла (МБ/MB) в соответствии с системным языком.
4.  **Кнопка выхода из аккаунта**
    - проверка видимости кнопки в офлайн‑режиме;
    - проверка появления кнопки после включения интернета.
5.  **Чат поддержки**
    - открытие чата через «Сообщить о проблеме» в настройках плеера;
    - авторизация внутри чата;
    - проверка открытия диалога с поддержкой.

### Ожидаемые результаты

- Поля ввода реагируют на нажатия, текст вводится корректно.
- При несовпадении ПИН‑кодов отображается ошибка «Пин‑коды не совпадают».
- Размер файла отображается на языке системы (например, «МБ» для русского, «MB» для английского).
- Кнопка выхода из аккаунта скрыта без интернета и появляется при восстановлении соединения.
- После авторизации через «Сообщить о проблеме» открывается чат с поддержкой.


    - размер файла иногда отображается на английском даже при неанглийском системном языке;
    - кнопка выхода остаётся видимой в редких офлайн‑сценариях;
    - чат поддержки не открывается после авторизации в определённых условиях.
