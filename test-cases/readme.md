# Test Cases

В этом разделе размещены тест-кейсы, составленные в процессе тестирования сервиса IVI.  
Документация будет дополняться и структурироваться по мере расширения проекта.

---

## Что содержится в папке

- Функциональные тест-кейсы основных разделов сервиса
- Проверка авторизации и регистрации (если применимо)
- Проверка каталога, карточек фильмов и плеера
- Негативные сценарии
- Проверка UI и отображения элементов
- Проверка стабильности навигации

---

## Формат тест-кейсов

Каждый тест-кейс оформлен в соответствии с базовыми требованиями:

- **ID**  
- **Название**  
- **Предусловия**  
- **Шаги**  
- **Ожидаемый результат**  
- **Фактический результат** (если нужен)  
- **Статус**  
- **Прикреплённые материалы** (скриншоты, видео, HAR-файлы)

---

## Примечание

Если в будущем появятся разделы (например, *player-tests*, *auth-tests*, *catalog-tests*), они будут вынесены в отдельные подпапки для удобства навигации.
