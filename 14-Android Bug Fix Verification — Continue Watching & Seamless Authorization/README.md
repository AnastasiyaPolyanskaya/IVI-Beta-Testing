# 14-Android Bug Fix Verification — Continue Watching & Seamless Authorization

Практический проект по верификации исправленных дефектов в мобильном приложении IVI (Android).

Проект направлен на проверку корректности работы функционала после внесения багфиксов, а также на подтверждение отсутствия регрессии в смежных пользовательских сценариях.

---

## Цель проекта

Подтвердить корректность исправлений и убедиться, что:

- контент корректно удаляется из блока «Продолжить просмотр»;
- работает бесшовная авторизация при переходе из мобильной веб-версии в приложение;
- не возникает регрессионных дефектов.

---

## Объект тестирования

- Платформа: Android  
- Тип тестирования: Manual QA  
- Формат: Bug Fix Verification / Regression Testing  

---

## Проверяемые исправления

### 1. Исправление удаления контента из блока «Продолжить просмотр»

**Задача:** ANDROID-26253  

**Проблема до исправления:**  
Контент не удалялся из блока «Продолжить просмотр» на главной странице старого формата.

**Проверка:**
- вызвать контекстное меню;
- нажать на кнопку удаления;
- убедиться, что контент удалён из блока.

**Ожидаемый результат:**  
Контент полностью удаляется из блока «Продолжить просмотр».

---

### 2. Исправление бесшовной авторизации (Web → App)

**Задача:** ANDROID-26569  

**Проблема до исправления:**  
При переходе из мобильной веб-версии ivi.ru в приложение авторизация не передавалась.

**Проверка:**
- пользователь авторизован в мобильной веб-версии;
- установленное приложение не авторизовано;
- в веб-версии нажать «Открыть в приложении»;
- происходит переход в приложение;
- проверить автоматическую авторизацию.

**Ожидаемый результат:**  
После перехода пользователь автоматически авторизуется в приложении без повторного ввода данных.
