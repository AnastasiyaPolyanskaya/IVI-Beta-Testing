# 14-Android Bug Fix Verification - Continue Watching & Seamless Authorization

Практический проект по верификации исправленных дефектов в мобильном приложении IVI (Android).

Проект направлен на подтверждение корректности работы блока «Продолжить просмотр» и механизма бесшовной авторизации при переходе из мобильной веб-версии в приложение.

---

## Цель проекта

Проверить корректность багфиксов и убедиться, что:

- контент корректно удаляется из блока «Продолжить просмотр» на главной странице;
- работает бесшовная авторизация при переходе Web → App;
- отсутствует регрессия смежного функционала.

---

## Объект тестирования

- Платформа: Android  
- Устройство: Redmi 9T  
- Версия ОС: Android 12.0.5.0 (QJQRXM)  
- Приложение: IVI  
- Тип тестирования: Manual QA  
- Формат: Bug Fix Verification / Regression Testing  

---

## Проверяемые сценарии

### 1. Удаление контента из блока «Продолжить просмотр»

Проверка выполнена в детском профиле на главной странице приложения.

Сценарий включает:
- наличие контента в блоке ПП;
- вызов контекстного меню;
- удаление контента;
- проверку фактического удаления.

Ожидаемый результат:  
Контент полностью удаляется из блока «Продолжить просмотр».

Статус: Passed

---

### 2. Бесшовная авторизация (Web → App)

Проверка передачи авторизации из мобильной веб-версии ivi.ru в установленное приложение.

Сценарий включает:
- авторизацию пользователя в веб-версии;
- переход в приложение через кнопку «Открыть в приложении»;
- проверку автоматической авторизации без повторного ввода данных.

Ожидаемый результат:  
Пользователь автоматически авторизуется в приложении.

Статус: Passed

---

## Результат

В ходе тестирования подтверждена корректность исправлений:

- удаление контента из блока «Продолжить просмотр» работает стабильно;
- бесшовная авторизация Web → App функционирует корректно.

Проект демонстрирует навыки верификации багфиксов, регрессионного тестирования и проверки кросс-платформенных пользовательских сценариев.
