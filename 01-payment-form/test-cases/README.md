# Тест-кейсы - Проект №1 (Форма оплаты)

В данном разделе представлены тест-кейсы, составленные в рамках исследовательского и функционального тестирования формы оплаты сервиса IVI.

## Содержимое
- **Тест-кейсыIVI.xlsx** — набор тестовых сценариев для проверки отображения полей, реакции UI на ввод реквизитов, поведения формы при невалидных данных, а также проверки визуального изменения цвета банковской карты.

## Применимость
Используется для Web и Mobile тестирования страницы оформления подписки / пробного периода.
