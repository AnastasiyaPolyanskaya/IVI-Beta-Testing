# Баг-репорты - Проект №1 (Форма оплаты)

В данном разделе собраны баг-репорты, найденные во время исследовательского и функционального тестирования формы оплаты сервиса IVI.

## Содержимое
- **Баг-репортыИВИ.docx** — документ с ошибками, обнаруженными при вводе реквизитов различных банков, проверке визуального отображения карты и валидации формы.

## Основные найденные дефекты
- Карта Т-Банк не окрашивается в желтый цвет.
- Иногда отсутствует визуальное обновление UI при вводе реквизитов.

## Применимость
Материалы относятся к Web и Mobile тестированию страницы оформления подписки / пробного периода.
