# Проект №1 - IVI: Тестирование формы оплаты (Web + Mobile)

## Описание
Проведено исследовательское тестирование формы оплаты на сайте и в мобильной версии IVI.
Основное внимание уделено корректности отображения полей, реакциям UI при вводе реквизитов различных банков, а также визуальному обновлению цвета карты.

## Окружение
- ОС: Windows 11, iOS 18.6.2  
- Устройства: ПК, iPhone 14 Pro Max  
- Браузеры: Chrome, Яндекс.Браузер  
- Тип тестирования: Exploratory (исследовательское)  
- Применимость: Web + Mobile  

## Проверенные сценарии
- Открытие формы оплаты на странице «Пробный период».  
- Проверка реакции UI на ввод реквизитов карт разных банков.  
- Проверка изменения цвета банковской карты (UI feature).  
- Проверка поведения формы при невалидных данных.  

## Найденные дефекты
- Карта Т-Банк не окрашивается в желтый цвет.  
- Иногда отсутствует визуальное обновление UI при вводе реквизитов.  

Формальные баг-репорты находятся в папке **bug-reports/**.
