# Баг-репорты - Проект №3 (Промокоды и подписки)

В данном разделе размещены баг-репорты, выявленные в процессе исследовательского и функционального тестирования промокодов, подписок и всплывающих окон в сервисе IVI.

## Основные направления проверок
Баги относятся к следующим областям:

- ввод промокодов и обработка ошибок  
- корректность пересчёта стоимости подписки  
- отображение статусов промокодов (валидный, невалидный, просроченный)  
- реакции UI на различные типы промокодов  
- корректность работы информационных всплывающих окон и подсказок  
- переходы в окно оформления подписки  
- отображение ошибок и предупреждений

## Содержимое
Файл баг-репортов включает:

- описание каждого выявленного дефекта  
- шаги воспроизведения  
- ожидаемый результат  
- фактический результат  
- окружение и уровень влияния на пользователя  

## Применимость
Материалы используются для анализа стабильности и корректности работы функционала промокодов и подписок в Web, iOS и Android версиях IVI.
