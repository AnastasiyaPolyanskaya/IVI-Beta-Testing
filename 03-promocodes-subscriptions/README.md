# Проект №3 - IVI: Промокоды, подписки и всплывающие окна (Web + Mobile)

## Описание
Проведено исследовательское и функциональное тестирование механики промокодов, оформления подписки и работы всплывающих подсказок и информационных окон в сервисе IVI.

Проверка выполнялась как в веб-версии, так и в мобильных приложениях (iOS/Android), с фокусом на корректность применения промокодов, обработку ошибок и отображение UI-элементов.

## Окружение
- ОС: Windows 11, iOS 18.6.2, Android 14  
- Платформы: Web, iOS-приложение, Android-приложение  
- Браузеры: Chrome, Яндекс.Браузер  

## Проверенные сценарии
- ввод промокода на странице подписки  
- отображение статуса промокода (валидный/невалидный/ошибка)  
- изменение стоимости подписки после применения промокода  
- реакция UI на неверный или просроченный промокод  
- отображение всплывающих подсказок и предупреждающих окон  
- переход в окно оформления подписки  
- проверка поведения формы при ошибках оплаты  

Формальные баг-репорты будут размещены в папке **bug-reports/**.
