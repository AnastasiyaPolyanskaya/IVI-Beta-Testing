# Тест-кейсы - Проект №3 (Промокоды и подписки)

В данном разделе размещены тест-кейсы, составленные для проверки работы промокодов, механики подписки и связанных всплывающих окон в сервисе IVI.

## Содержимое
Файл тест-кейсов включает сценарии для проверки следующих функций:

- ввод промокода на странице оформления подписки  
- обработка валидных и невалидных промокодов  
- корректность пересчёта стоимости подписки  
- отображение статусов промокода (успешно / ошибка / просрочен)  
- проверка логики применения промокода при разных условиях  
- корректность отображения всплывающих подсказок и информационных окон  
- проверка переходов в окно подписки  

## Применимость
Тест-кейсы относятся к Web, iOS и Android версиям сервиса IVI и используются для функционального и UI/UX тестирования сценариев, связанных с промокодами и оформлением подписки.
