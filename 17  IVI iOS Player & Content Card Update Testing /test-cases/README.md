## Test Cases

В данной папке представлены тест-кейсы, разработанные для проверки обновлений и исправлений в мобильном приложении **IVI** для платформы **iOS**.

Тестирование направлено на проверку стабильности приложения, корректности работы плеера, карточки контента, а также воспроизведения различных типов контента после внесённых изменений.

---

## Цель тестирования

- Подтвердить исправление ранее выявленных дефектов;
- Проверить корректность работы обновлённого плеера;
- Убедиться в стабильности приложения при работе с коллекциями и контентом;
- Исключить появление регрессии в ключевых пользовательских сценариях.

---

## Покрываемые области тестирования

### 1. Работа с коллекциями (покупки)

Проверяется исправление краша при открытии ранее приобретённых коллекций.

Проверяемые сценарии:
- переход в раздел «Покупки»;
- открытие ранее купленных коллекций.

Ожидаемый результат — коллекции открываются без вылетов приложения.

---

### 2. Обновлённый плеер

Проверяется работа плеера после обновления внутренних компонентов.

Проверяемые сценарии:

- воспроизведение контента;
- выбор качества (Auto и фиксированные значения);
- выбор локализации;
- воспроизведение рекламы (для пользователей без подписки);
- работа режима **Picture-in-Picture (PiP)**.

---

### 3. Кастинг (передача контента)

Дополнительно проверяется возможность трансляции контента на внешние устройства:

- Android TV;
- Smart TV.

Ожидаемый результат — воспроизведение на внешних устройствах работает корректно.

---

### 4. Карточка контента

Проверяется обновлённая карточка контента.

Проверяемые сценарии:
- корректность отображения карточки;
- работа всех кнопок и элементов интерфейса.

---

### 5. DRM-контент

Проверяется воспроизведение защищённого контента.

Ожидаемый результат — контент воспроизводится без ошибок.

---

### 6. Live Activity

Проверяется отображение виджета активности при воспроизведении контента.

Проверяемые сценарии:
- просмотр контента более 1 минуты;
- блокировка устройства или открытие системной шторки.

Ожидаемый результат — отображается Live Activity виджет.

---

## Структура тест-кейсов

Каждый тест-кейс содержит:

- ID тест-кейса  
- Название  
- Предусловия  
- Шаги воспроизведения  
- Ожидаемый результат  
- Фактический результат  
- Окружение  
- Статус выполнения  

---

## Окружение тестирования

- Устройство: **iPhone 14 Pro Max**
- Платформа: **iOS**
- Приложение: **IVI**
